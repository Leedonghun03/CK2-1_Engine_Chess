# CK2-1_Engine_Chess
청강대 2인 엔진 팀플 과제<br/>
기말 과제 내용으로 만든 게임은 멀티플레이 체스 게임을 제작했습니다.

작업 담당은 다음과 같습니다.
- 이동훈
1. 인게임 맵 제작
2. 유령이 체스말 들기/놓기
3. 체스말 기본 이동
4. 이동 가능 위치 GUI 표시
5. 앙파상
6. 캐슬링
7. 프로모션
8. 체크메이트
<br/>

- 최주환
1. 로그인 제작
2. 로비 제작
3. 인게임 이동 로직 (플레이어, 게임 상태 동기화)
4. 클라이언트 연결 작업
<br/>

- 공동 작업 <br/>
1. Git, Fork 연동
2. 체스 기물 파괴 Mesh
3. 체스 기물 파괴 셰이더 구현
<br/>
